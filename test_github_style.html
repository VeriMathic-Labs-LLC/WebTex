DIN Model Equations

The attention weight is:

$$a(u, i) = \frac{\exp(z(u, i))}{\sum_{j \in \{h_1, h_2, ..., h_H\}} \exp(z(u, j))}$$

Where $z(u, i) = v_u^T A v_i + b$ is the attention score.

Inline Math Examples

The model uses $D \times K$ dimensional embeddings where $D$ is the embedding dimension and $K$ is the number of features.

Block Math Example

$$e_i = pooling(e_{i1}, e_{i2}, ...e_{ik})$$

Complex Example from the GitHub page

$$L=-\frac{1}{KN} \sum_{(x,y) \in \mathcal{S}} (y \log p(x) + (1-y) \log (1-p(x)))$$
